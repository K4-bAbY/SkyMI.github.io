# 常见问题解答 (FAQ)

## 安装与兼容性

### Q: 我的设备支持 SkyMI 内核吗？
**A:** 目前 SkyMI 内核主要支持 Xiaomi Redmi 12 5G (代号: `sky`)。我们正在逐步扩展对其他设备的支持。请定期查看项目更新以获取最新支持列表。

### Q: 如何确认我的设备是否为 Redmi 12 5G？
**A:** 
1. 进入 **设置** → **关于手机** → **设备名称**
2. 查看设备代号是否为 `sky`
3. 或在 ADB 中执行：`adb shell getprop ro.product.device`

### Q: 不同 Android 版本的设备可以使用同一个内核吗？
**A:** 不可以。SkyMI 内核针对特定的 Android 版本进行优化。请确保下载与您设备 Android 版本相匹配的内核版本。

## 刷入过程

### Q: 刷入前需要做什么准备？
**A:** 
1. **备份数据**：完整备份您的设备数据（照片、文件、应用等）
2. **解锁 Bootloader**：如果尚未解锁，请先解锁
3. **安装 Recovery**：安装 TWRP 或 OrangeFox Recovery
4. **下载内核**：从官方渠道获取最新的 SkyMI 内核 ZIP 包
5. **验证文件**：检查 SHA256 校验和以确保文件完整性

### Q: 解锁 Bootloader 会发生什么？
**A:** 解锁 Bootloader 会：
- 清除设备上的所有数据（包括应用、设置、文件等）
- 允许您刷入自定义 Recovery 和内核
- 可能影响设备保修

**警告**：解锁前请备份所有重要数据。

### Q: 如何进入 Recovery 模式？
**A:** 
1. 关闭设备
2. 同时按住 **音量减键** + **电源键**
3. 进入 Fastboot 模式后，选择 Recovery 选项
4. 或者在 ADB 中执行：`adb reboot recovery`

### Q: 刷入过程中设备断电了怎么办？
**A:** 这可能导致设备变砖。如果发生这种情况：
1. 连接到电源
2. 进入 Recovery 模式
3. 尝试重新刷入内核
4. 如果仍无法解决，请寻求专业帮助或联系社区

### Q: 刷入后首次启动需要多长时间？
**A:** 首次启动可能需要 5-10 分钟。这是正常的，系统在优化应用。请耐心等待，不要中断过程。

## 刷入后的问题

### Q: 刷入内核后设备无法启动怎么办？
**A:** 
1. **进入 Recovery**：按住音量减键 + 电源键
2. **清除缓存**：在 Recovery 中选择清除 Dalvik/Cache
3. **重新启动**：选择重启系统
4. **如果仍无法启动**：
   - 进入 Recovery
   - 刷入官方 Boot 镜像以恢复
   - 或重新刷入 SkyMI 内核

### Q: 如何验证内核是否成功刷入？
**A:** 
1. **在系统中**：设置 → 关于手机，查看内核版本
2. **使用 ADB**：执行 `adb shell uname -a`
3. **使用终端应用**：在设备上打开终端，执行 `uname -a`

### Q: 刷入后性能没有改善，怎么办？
**A:** 
1. **清除缓存**：进入 Recovery 清除 Dalvik/Cache
2. **重启设备**：完全关闭后重启
3. **等待优化**：系统需要时间来优化应用
4. **检查设置**：确保没有启用不必要的功能
5. **查看日志**：使用 logcat 检查是否有错误

### Q: 内核会导致设备发热吗？
**A:** SkyMI 内核经过优化，不应导致过度发热。如果您发现设备异常发热：
1. 检查是否有后台应用占用 CPU
2. 清除应用缓存
3. 在 Recovery 中清除 Dalvik/Cache
4. 如果问题持续，考虑回退到官方内核

## 功能与性能

### Q: SkyMI 内核提供哪些性能改进？
**A:** SkyMI 内核提供以下改进：
- **CPU 调度优化**：更快的任务响应
- **I/O 优化**：更快的应用启动和数据加载
- **内存优化**：更好的多任务处理
- **功耗优化**：更长的电池续航

### Q: 内核支持超频吗？
**A:** SkyMI 内核本身不包含超频功能，但您可以使用第三方工具（如 Kernel Adiutor）来调整 CPU 频率。请谨慎操作，以避免设备损伤。

### Q: 可以自定义内核参数吗？
**A:** 是的，SkyMI 内核提供多个可调参数。您可以使用以下工具进行调整：
- **Kernel Adiutor**：图形化界面
- **Termux**：命令行工具
- **设置应用**：某些参数可在系统设置中调整

### Q: 内核会影响游戏性能吗？
**A:** SkyMI 内核的优化应该会改善游戏性能，包括：
- 更低的帧率波动
- 更快的加载时间
- 更稳定的帧率

## 回退与恢复

### Q: 如何回退到官方内核？
**A:** 
1. **进入 Recovery**：按住音量减键 + 电源键
2. **刷入官方 Boot 镜像**：选择您备份的原始 Boot 分区
3. **重启系统**：系统将使用官方内核启动

### Q: 如果我没有备份原始 Boot 分区怎么办？
**A:** 
1. 从小米官方网站下载您设备的官方 ROM
2. 从 ROM 中提取 Boot 镜像
3. 在 Recovery 中刷入该镜像

### Q: 回退会删除数据吗？
**A:** 仅刷入 Boot 镜像不会删除数据。但为了安全起见，建议在刷入前备份数据。

## 保修与安全

### Q: 刷入 SkyMI 内核会影响保修吗？
**A:** 这取决于您的设备制造商的保修政策。通常：
- 刷入自定义内核可能会影响官方保修
- 小米可能会拒绝为已刷入自定义内核的设备提供保修服务
- 您可以通过刷入官方内核来恢复保修资格

### Q: SkyMI 内核是否安全？
**A:** SkyMI 内核基于官方 Linux LTS 内核，定期合入上游安全补丁。但与任何自定义软件一样，使用自定义内核存在一定风险。请：
- 仅从官方渠道下载内核
- 验证文件完整性
- 定期更新到最新版本

### Q: 如何报告安全问题？
**A:** 如果您发现安全漏洞，请：
1. 不要在公开渠道讨论
2. 通过 GitHub 私密问题或电子邮件联系开发者
3. 提供详细的漏洞描述和复现步骤

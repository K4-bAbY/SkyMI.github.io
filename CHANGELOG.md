## 版本号说明

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范：

- **主版本号 (Major)**：不兼容的 API 变更或重大功能变更
- **次版本号 (Minor)**：向后兼容的功能新增
- **修订号 (Patch)**：向后兼容的 Bug 修复
- **预发布版本**：使用 `-alpha`、`-beta`、`-rc` 等后缀

示例：
- `1.0.0-beta1`：第一个测试版本
- `1.0.0-rc1`：第一个候选发布版本
- `1.0.0`：正式版本

# SkyMI Android 内核项目

## 欢迎

欢迎来到 **SkyMI Android 内核项目**！这是一个致力于为兼容 Android 设备提供优化、稳定且高性能内核的开源项目。

SkyMI 内核基于 Linux LTS 内核，集成了多项性能优化、功耗管理和稳定性增强，为您的设备带来更流畅的用户体验。

## 项目概览

### 核心特性

SkyMI 内核提供以下优化和特性：

#### 🚀 性能优化
- **CPU 调度器优化**：采用 EAS（Energy-Aware Scheduling）和 PELT 优化，提升任务响应速度
- **I/O 优化**：集成高效的 I/O 调度器（FIOPS、Maple），加速应用启动和数据加载
- **内存管理**：优化 ZRAM/ZSWAP 配置，提高多任务处理能力

#### 🔋 续航增强
- **电源管理**：精细化 CPU/GPU 频率控制，优化空闲状态功耗
- **唤醒管理**：减少不必要的唤醒源，防止后台应用耗电

#### 🛡️ 稳定性与兼容性
- **上游合入**：定期合入 Linux LTS 内核最新补丁，确保安全性
- **平台兼容性**：针对特定 Android 版本进行优化

#### ⚙️ 额外功能
- **快充支持**：优化快充协议支持
- **自定义调校**：提供内核参数调校选项

## 刷入指南

### ⚠️ 重要警告

刷入自定义内核存在风险，可能导致设备变砖或数据丢失。请务必：
- 备份您的设备数据
- 了解刷入过程中的每一步
- 在遇到问题时能够恢复官方 Boot 镜像

### 刷入步骤

1. **解锁 Bootloader**
   - 您的设备必须已解锁 Bootloader
   - 解锁过程会清除设备数据

2. **安装自定义 Recovery**
   - 推荐使用 TWRP 或 OrangeFox Recovery
   - 确保 Recovery 版本与您的设备兼容

3. **下载内核包**
   - 从官方发布渠道获取最新版本的 SkyMI 内核 ZIP 包
   - 验证 SHA256 校验和以确保完整性

4. **进入 Recovery 模式**
   - 关闭设备
   - 按住特定按键组合进入 Recovery（通常为音量键 + 电源键）

5. **备份分区**
   - 在 Recovery 中备份 `Boot`、`Vendor Boot`、`DTBO` 分区
   - 将备份文件导出到安全位置

6. **刷入内核**
   - 选择下载的内核 ZIP 包进行刷入
   - 等待刷入完成

7. **清除缓存（可选但推荐）**
   - 刷入后清除 Dalvik/Cache
   - 这有助于避免启动问题

8. **重启系统**
   - 重启设备进入系统
   - 首次启动可能需要较长时间

## 常见问题

### Q: 我的设备支持 SkyMI 内核吗？
**A:** 目前主要支持 Xiaomi Redmi 12 5G (sky)。请关注项目更新以获取最新支持列表。

### Q: 刷入内核后遇到问题怎么办？
**A:** 
- 尝试重新刷入官方 Boot 镜像
- 检查 logcat 日志以诊断问题
- 在相关社区论坛或 GitHub Issues 寻求帮助

### Q: 如何验证内核是否成功刷入？
**A:** 
- 进入系统设置 → 关于手机
- 查看内核版本是否显示为 SkyMI
- 使用 `uname -a` 命令查看内核信息

### Q: 可以回退到官方内核吗？
**A:** 是的，您可以通过刷入官方 Boot 镜像来回退。请确保您在刷入前已备份原始 Boot 分区。

### Q: SkyMI 内核会影响保修吗？
**A:** 这取决于您的设备制造商的保修政策。通常，刷入自定义内核可能会影响保修。
